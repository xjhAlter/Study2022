# 七、列表

符号跟内容之间要有空格

## 无序列表

-   列表内容
-   列表内容
-   列表内容

## 有序列表

1. 列表内容
2. 列表内容
3. 列表内容

## 列表嵌套

上一级和下一级之间敲三个空格

-   列表内容
    -   列表内容
    -   列表内容
